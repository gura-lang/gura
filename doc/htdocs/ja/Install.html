インストール方法

Windows へのインストール方法

インストーラ gura-0.3.0-win32.msi を実行してください。拡張子 .gura, .guraw, .gurc および .gurcw が実行ファイルに関連付けられます。

Linux へのインストール方法

Ubuntu や Fedora は、デフォルトでは C++ コンパイラや readline ライブラリが入っていないので、以下のようにパッケージをインストールしておきます。

Ubuntu の場合:

$ sudo apt-get install g++ libreadline5-dev

Fedora の場合:

# yum install gcc gcc-c++ readline-devel

ソースパッケージ gura-0.3.0.tar.gz をダウンロードし、以下の手順で Gura の本体をビルド・インストールします。 

$ tar xvfz gura-0.3.0.tar.gz
$ cd gura-0.3.0/src
$ ./configure
$ make
$ sudo make install
$ sudo ldconfig     # 最初のインストールの時

続けて、以下の手順でモジュールファイルをビルド・インストールします。 

$ ./build_modules.gura
$ sudo ./build_modules.gura install

エラーが出る場合は、モジュールに必要なライブラリがシステムにインストールされていない可能性があります。エラーメッセージにしたがって、パッケージをインストールしてください。
