\section{\class{wxXmlProperty}}\label{wxxmlproperty}

Represents a node property.

Example: in {\tt <img src="hello.gif" id="3"/>}, {\tt ``src"} is property with value
{\tt ``hello.gif"} and {\tt ``id"} is a property with value {\tt ``3"}.

No base class
